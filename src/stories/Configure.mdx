# InstantUI React 组件库

InstantUI 是一个基于 React 的 UI 组件库，提供了丰富的 UI 组件和工具，帮助开发者快速构建美观、易用的用户界面。

## 📚 组件文档结构

每个组件包含：- 基础用法示例 - Props类型定义表 - 交互式Playground - 设计规范参考

## 🛠 InstantUI 集成配置

### 1. 安装必要依赖

```bash
npm install instantui --save-dev
```
